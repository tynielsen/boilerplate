CSS3/Sass & beyond!

With the latest CSS updates that have been released, applying styles are significantly easier, and have support from all modern browsers.

Here Are Some Examples:


/** *******************
  Variables:
******************** */
::root {
  --primary-color: hotpink;
  --secondary-color: yellow;
  --bg1: #00FF00;
  --bg2: #000000;
}

/** *******************
  Nesting:
******************** */
header {
  background-color: var(--bg2);
  color: var(--secondary-color);
  
  p { 
    color: var(--secondary-color);
  }
}

// Easy-to-use layout options exist now (for the past few years), so no more hacking floats…
